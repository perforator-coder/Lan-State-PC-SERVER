  <h1 align="center">
    Lan State PC SERVER
  </h1>
  <p align="center">
    Это программа пердназначеная для управления пк (выключение и перезагрузка) и получения его данных по локальной сети.
      <p align="center">
     <a href="https://github.com/perforator-coder/Lan-State-PC-CLIENT">
    Клиенская часть
  </a>
    </p>
  </p>
  <h1 align="center">
    Какие данные получает программа от клиента?
  </h1>
  <ul>
    <li>
      Полное имя OC и редакция
    </li>
    <li>
      Локальный ip-адрес
    </li>
    <li>
      MAC-адресс
    </li>
    <li>
      Полное имя CPU
    </li>
    <li>
      Полное имя GPU
  </ul>
  <h1 align="center">
    Что может сделать сервер с клиентом?
  </h1>
  <ul>
    <li>
      Отправить сообщение клиенту
    </li>
    <li>
      Отправить пк клиента на пренудтельную перезагрузку
    </li>
    <li>
      Отправить пк клиента на пренудительное завершение работы
    </li>
  </ul>
  <sub align="center">
    Данное приложение было разработано для прохождения практики в коледже 
  </sub>
